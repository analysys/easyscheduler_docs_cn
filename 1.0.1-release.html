1.0.1 release

Easy Scheduler Release 1.0.1

Easy Scheduler 1.0.2是1.x系列中的第二个版本。更新内容具体如下：

  • 1,outlook TSL 发邮件支持
  • 2,servlet 和 protobuf jar冲突解决
  • 3,创建租户同时建立linux用户
  • 4,重跑时间负数
  • 5,单机和集群都可以使用install.sh一键部署
  • 6,队列支持界面添加
  • 7,escheduler.t_escheduler_queue 增加了create_time和update_time字段
